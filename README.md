## ğŸŒ English Description

### Key Features:
*   **Flexible Age Input:** Calculate growth based on days, months, or years.
*   **Corrected Age for Preterm Babies:** Specifically designed for premature infants (born before 37 weeks) to adjust their growth metrics up to 2 years of age.
*   **Advanced Metrics:** Includes Weight-for-Age, Height-for-Age, and **BMI-for-Age** analysis.
*   **Interactive Visual Charts:** Powered by `Chart.js`, it plots the child's data against the 3rd, 50th (Median), and 97th percentiles.
*   **Single-File Architecture:** Runs entirely in any modern browser without the need for a backend or installation.
*   **Responsive Design:** Optimized for both desktop and mobile devices.

### How to Use:
1.  Clone this repository or download the `index.html` file.
2.  Open the file in any modern web browser (Chrome, Firefox, Safari, Edge).
3.  Enter the child's details and click on the analysis buttons.

### Technologies:
*   HTML5 & CSS3 (Custom Styles)
*   Vanilla JavaScript
*   [Chart.js](https://www.chartjs.org/) for data visualization
